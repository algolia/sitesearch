AskAI Sidepanel lets you add conversational answers to any page without redesigning the core layout. Think contextual help, onboarding guides, or product discovery tucked into a drawer.

## Use cases

- Documentation sites that need contextual Q&A
- SaaS dashboards where users want quick explanations without leaving the flow
- Ecommerce PDPs that benefit from conversational sizing or compatibility tips

## Usage

```tsx
import { Sidepanel } from '@algolia/sidepanel-react';
import '@algolia/sidepanel-react/dist/sidepanel.css';

<Sidepanel
  config={{
    applicationId: 'YOUR_APP_ID',
    apiKey: 'YOUR_SEARCH_API_KEY',
    indexName: 'YOUR_INDEX_NAME',
    assistantId: 'YOUR_ASSISTANT_ID',
  }}
  suggestions={[
    'Show me trending items',
    'Explain how billing works',
    'Find docs about webhooks',
  ]}
/>
```
